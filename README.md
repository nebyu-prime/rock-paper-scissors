# Rock Paper Scissors

A tiny browser game: pick Rock, Paper, or Scissors and play against the computer.

## Quick Start
- Open `index.html` in any modern browser.
- Or from a terminal (Windows):

```bash
explorer.exe index.html
```

## Files
- `index.html`: UI markup
- `script.js`: game logic

## Tech
Plain HTML + JavaScript. No build step or dependencies.
